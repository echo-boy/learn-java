## Java基础知识

### 数据类型

#### 基本数据类型

    Java提供了八种基本类型
        六种数字类型（四个整数型，两个浮点型）
            byte
            short
            int
            long
            float
            double
        一种字符类型
            char
        一种布尔型
            boolean

#### 引用类型

    对象

### 变量定义

变量：相对于常量，表示不确定的值或对于值的一个引用符号。可以代表某个基本数据类型或者指向一个对象。

> Java中变量必须先定义后使用

定义:

    type identifier [ = value][, identifier [= value] ...] ;

    // 变量可以赋值，然后可以之后再修改其值即重新赋值。
